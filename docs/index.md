# 🎬 CineSparks AI Docs

> **실시간 시네마 데이터와 초거대 Gemini API의 정교한 시너지**

CineSparks AI 프로젝트의 공식 가이드 및 기술 설계 문서 포털에 오신 것을 환영합니다. 본 사이트는 GitHub Pages (Jekyll) 및 최적의 공식 플러그인들을 탑재하여 실시간 검색 색인 최적화(SEO)와 무결점 빌드를 지원합니다.

---

## 🛠️ 프로젝트 빌드 및 특징

* **실시간 박스오피스 TOP 10**: 한국영화진흥위원회(KOBIS) API 연동
* **TMDB 비주얼 에셋**: 영화 포스터, 와이드 백드롭, 평점, 한국어 줄거리 자동 병합
* **유튜브 공식 예고편**: 16:9 반응형 고해상도 다이렉트 플레이어 탑재
* **Gemini AI 평론 파트너**: 한 줄 소감 기반 전문 비평문 자동 확장(RAG)

---

## 📦 탑재된 Jekyll 플러그인 목록

본 문서 사이트는 다음과 같은 깃허브 공식 화이트리스트 플러그인을 탑재하여 무인 배포됩니다:

1. **`jekyll-seo-tag`**: 검색엔진 노출 지수 최적화 및 메타데이터 자동 삽입
2. **`jekyll-sitemap`**: Google/Naver 검색 봇 색인용 `sitemap.xml` 자동 갱신
3. **`jekyll-feed`**: 사이트 소식 전파용 `feed.xml` RSS 피드 자동 발행

---

## 🚀 빠른 시작 가이드

로컬 컴퓨터에서 본 문서 사이트를 직접 실행하려면 루비(Ruby) 환경에서 아래와 같이 구동합니다:

```bash
cd docs
bundle install
bundle exec jekyll serve
```

웹 브라우저를 열고 `http://localhost:4000/cinesparks-ai/` 로 접속하시면 실시간 렌더링을 확인하실 수 있습니다.
